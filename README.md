A high-fidelity landing page clone of the official Apple website. This project replicates Apple's iconic minimalist design, premium aesthetic, and smooth user interface using vanilla web technologies.

**🔗 Live Demo:** [https://appyclone.netlify.app/]

---

## ✨ Key Features

*   **Premium UI/UX:** Replicates Apple’s clean layout, focusing on whitespace, bold typography, and high-quality imagery.
*   **Responsive Multi-page Design:** Includes dedicated sections for various products and categories.
*   **Custom Typography:** Integration of specific brand-aligned fonts for an authentic look and feel.
*   **Motion-Enhanced Interface:** Smooth transitions and hover effects consistent with modern design trends.

## 🛠️ Technical Stack

*   **Frontend:** HTML5, CSS3 (Flexbox & Grid), JavaScript (Vanilla ES6+).
*   **Visuals:** Optimized image assets and custom web fonts for high-resolution displays.
*   **Deployment:** Designed for seamless hosting via platforms like Netlify.

## 📂 Project Structure

As seen in **image_90287d.png**, the project is organized into modular directories:
```text
├── home.html           # Main Landing Page (entry point)
├── css/                # Custom stylesheets for layout and design
├── font/               # Custom web font files
├── html/               # Additional sub-pages and sections
├── images/             # Product imagery and high-res assets
├── js/                 # JavaScript logic for UI interactions
└── thumnaillmg.jpg     # Project preview/thumbnail image
```
## 🚀 Local Setup

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/aravinthsenthil23/Apple.git](https://github.com/aravinthsenthil23/Apple.git)
    ```
2.  **Navigate to the folder:**
    ```bash
    cd Apple
    ```
3.  **Run the project:**
    Open `home.html` in your browser. For the best experience, use the **Live Server** extension in VS Code.
